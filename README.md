# Rabbit News

Rabbit News is a Rabbit r1 creation that turns the device into a compact news hub.

It is designed for quick headline browsing, simple on-device reading, and smooth navigation with the Rabbit wheel-style interaction model.

## Live Links

- App: https://badrazzam9.github.io/rabbit-news/
- Install page: https://badrazzam9.github.io/rabbit-news/install.html
- Rabbit install QR: https://badrazzam9.github.io/rabbit-news/rabbit-news-r1-install-qr.png
- Worker API: https://rabbit-news-worker.swordandscroll.workers.dev

## What It Does

- Shows fresh breaking news on the home screen
- Lets you browse regional news feeds
- Lets you search news by topic or country
- Opens readable article views when extraction works cleanly
- Falls back to the original source when a page is not reader-friendly

## Main Features

- 3D wheel-style headline browsing
- Fresh breaking feed on launch and refresh
- Regional browsing
- Search across multiple news sources
- Article extraction with graceful fallback
- Adjustable reading font sizes
- Rabbit-friendly compact layout

## Controls

- Scroll: move through headlines
- Tap: open the selected card
- Refresh button: refresh the current context
- Home button: return to the home screen
- Side button: go back from cards or article view
- Font controls in article view: increase or decrease text size

## Install On Rabbit r1

1. Open the install page or the QR image link.
2. Scan the QR code from the Rabbit r1 creations flow.
3. Install the creation.
4. Launch Rabbit News from your creations list.

## Notes

- This project is built as a hosted Rabbit creation.
- News freshness and article quality depend on upstream publishers and feeds.
- Some publisher pages will still open in source mode instead of clean reader mode when extraction is not reliable.
